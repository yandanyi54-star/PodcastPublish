## 紧急回滚方案

如果发布后发现问题：

```bash
# 方式一：回退到上一个 tag
git tag                     # 查看当前 tag
git checkout tags/v0.9.0    # 切换到上一个稳定版本
npm run build               # 重新构建
# 部署 dist/ 目录即可

# 方式二：使用 dist 历史
# 每次构建前备份当前 dist/ 目录
```
