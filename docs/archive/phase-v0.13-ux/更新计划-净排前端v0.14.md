## 3. 工程纪律（每步协议）

```
vite build  →  零错误（managed node 22.22.2）
vitest run  →  31 tests pass（含 B4/B9 新增 smoke）
probe_verify.mjs  →  全绿（既有探针回归）
git commit  →  单任务一颗 commit，message 带断点号（如 "B4: publish wizard"）
```

- 功能类改动（B4/B5/B9）先写 **PRD**；架构决策类（B1 解耦、B5 三层模型）写 **ADR**；复用 `docs/templates/` 与 `docs/STANDARDIZATION.md`。
- 完成后更新 `CHANGELOG.md` 与 `RELEASE_CHECKLIST.md`（沿用 v0.13.x 风格，每条附验收）。
- 纯前端 / 不联网约束：发布向导不得引入任何服务端依赖；`openWeChat` 仅 `window.open(mp.weixin.qq.com)`。
