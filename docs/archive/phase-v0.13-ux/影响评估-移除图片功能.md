## 已存在的 Markdown 图片内容

用户已有的 `![](url)` Markdown 图片内容：
- **编辑器**：Milkdown 仍能正常渲染（`image` 节点在 schema 中仍存在）
- **预览**：`buildHtml` 仍正常渲染 `<img>`
- **复制/导出**：HTML 中的 `<img>` 标签正常

唯一变化：无法再对已有图片调宽度。已有宽度信息（`imageWidthMap` 在 localStorage 中）会被忽略。这符合「移除插图功能」的预期。
