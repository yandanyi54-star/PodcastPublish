## 验收建议（真机/模拟器）

- **iPhone SE (375px)**：底部栏单行横向滚动，12 个图标均可点；编辑区不被遮挡。
- **iPhone 横屏**：底栏变矮，编辑区可用，预览浮钮不挡内容。
- **iPad 竖屏 (810px)**：打开「样式/AI」面板时，预览自动收窄，编辑器保持可读宽度。
- **深色模式**：新增样式全用 CSS 变量，自动继承（无额外适配）。
