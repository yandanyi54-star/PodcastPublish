## M3 — 打磨 + 架构基座

### B6 上下文栏一致性

- [x] 上下文栏仅在「有 ≥1 个可用操作」时出现（图片节点不显示空栏）
- [x] 选中 <10 字短文：给出格式化操作（加粗/斜体/链接），不再空白死路
- [x] 标题/装饰块选中：保留各自操作

### B7 导入去向

- [x] 「从文本提纯」后有明确确认反馈（关闭面板 + 字数 toast）
- [x] 导入后跳转到编辑器顶部
- [x] 出现 toast「已导入 N 字」

### B11 AI 前置说明

- [x] 首次打开 AI 面板提示「需自备 API Key，仅本地直连厂商」
- [x] 提示不阻断操作，可关闭/不再打扰（localStorage 记忆）

### B12 复制 toast 条件化

- [x] 正文含 `data:` URI 时才警告 base64 图片问题
- [x] 仅用 `![alt](url)` 外链图的用户复制时**不**收到该告警

### B10 组件化 + tokens.css

- [x] 设计变量基座落地（`src/styles/variables.css`：语义色/圆角/阴影/过渡 + 补 `--accent`；计划原写 tokens.css，实际增强既有 variables.css）
- [x] 按组件树拆分（`PublishWizard`/`ThemeModeSwitch`/`AppearancePanel` 已独立；删孤儿 `StylePanel`/`SettingsPanel`）
- [ ] [~] `App.vue` 行数：因 M1–M3 新增功能净增（≈2800→3457），但组件边界清晰、职责分离达成（EntryCoordinator 未拆，可后续）

### B8 响应式收尾

- [ ] ≤768 / 769–1024 / ≥1025 三档行为清晰
